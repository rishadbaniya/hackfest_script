Usage :

 Note : It must have a field that represents the Devfolio URL, it must be "Submission URL" field

 How to use?

1) Run script_0.js as following :
 => node script_0.js path_to_raw_csv_file.csv


2) Run script_1.js as
=> node script_1.js

3) Run script_3.js as 
=> node script_3.js

The final output file will be created at the output folder, which will contain Qualified File which represents the teams that were qualified
and it also contains Disqualified File which contains the teams that were not qualified
